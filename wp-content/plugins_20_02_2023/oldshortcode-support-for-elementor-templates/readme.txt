== Description ==

<h2>Shortcode For Elementor Templates</h2>

Include any elementor template using shortcode. It will add shortcode column to existing elementor template admin columns. Use [el_shortcode id="235"] inside any page or post or template. 

inside template you can use <?php echo do_shortcode('[el_shortcode id="235"]');  ?>

do not forget to replace 235 with your elementor template id.
